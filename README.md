# Stayhub-Booking-App


<h1 align="center">🤖 AI Resume Builder</h1>

<p align="center">
An intelligent Resume Builder application powered by <strong>Java</strong> and <strong>Google Gemini AI</strong> that helps users create professional, ATS-friendly resumes effortlessly. The system leverages Generative AI to generate resume summaries, improve content quality, and provide personalized suggestions based on user input.
</p>

<h2>🚀 Features</h2>

<ul>
<li>📄 Create professional resumes in minutes</li>
<li>🤖 AI-powered content generation using Gemini AI</li>
<li>✨ Generate professional summaries automatically</li>
<li>🎯 ATS-friendly resume formatting</li>
<li>📝 Edit and customize resume sections</li>
<li>💼 Add Education, Skills, Projects, and Experience</li>
<li>📥 Download resumes in PDF format</li>
<li>🔍 Resume content optimization suggestions</li>
<li>📱 Responsive and user-friendly interface</li>
</ul>

<h2>⚙️ Installation</h2>

<pre>
# Clone the repository
git clone https://github.com/ELANTHENDRAL12/AI-Resume-Builder.git

# Navigate to the project directory
cd AI-Resume-Builder

# Build the project
mvn clean install

# Run the application
mvn spring-boot:run
</pre>

<h2>🔑 Configuration</h2>

<p>Add your Gemini API Key in <code>application.properties</code>:</p>

<pre>
gemini.api.key=YOUR_GEMINI_API_KEY
</pre>

<h2>🎯 How It Works</h2>

<ol>
<li>User enters personal and professional details.</li>
<li>Application sends the data to Gemini AI.</li>
<li>Gemini AI generates professional resume content.</li>
<li>User reviews and customizes the generated content.</li>
<li>Final resume can be downloaded and shared.</li>
</ol>
